Research & Projects

🤖 AI Chatbot

A production-ready AI chatbot built with Next.js, LangChain, and LangGraph.
Multi-model support · PDF RAG · Agentic workflows · Streaming responses

⚡ Next.js⚛️ React🔗 LangChain🕸️ LangGraph🌊 Tailwind CSS🤖 Open AI🧠 Anthropic🗺️ Open street Map🗄️ Vector Store🌍 Multi language support⚙️ Agentic workflows📄 PDF RAG

This project is a highly scalable, agentic AI Chatbot designed to handle complex queries using state-of-the-art Large Language Models. By leveraging LangChain and LangGraph, it transcends standard conversational bots, enabling true agentic workflows where the AI can intelligently route requests, utilize external tools, and fetch custom data via a Retrieval-Augmented Generation (RAG) pipeline.

🚀 Core AI Advantages & Best Features

  • 🧠
    True Agentic Workflows: Utilizes LangGraph for stateful, multi-step reasoning, allowing the AI to autonomously decide when and how to use external tools.
  • 📚
    Intelligent PDF RAG: Upload documents directly to the interface. The system chunks, embeds, and stores the text for highly accurate, context-aware answers.
  • ⚡
    Real-time Streaming: Delivers instantaneous, token-by-token streaming responses for a seamless user experience, avoiding long loading times.
  • 🔄
    Multi-Model Flexibility: Hot-swap between Anthropic Claude and OpenAI GPT models seamlessly to balance cost, speed, and reasoning capabilities.
  • 🛠️
    Extensible Architecture: Designed from the ground up to easily integrate APIs, custom databases, or web search capabilities without breaking existing logic.
